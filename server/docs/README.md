# 团队报名系统服务端文档

本目录包含团队报名系统服务端的所有文档。

## 当前项目状态

- 基础架构搭建完成，采用 Express 框架
- 依赖注入系统实现，使用容器管理服务实例
- 账户领域基本功能实现
- 性能监控、错误处理、健康检查等机制

## 项目结构

```
src/
├── app.js               # 应用入口
├── domains/             # 领域层
│   ├── BaseService.js   # 服务基类
│   └── account/         # 账户领域
├── infrastructure/      # 基础设施层
│   ├── common/          # 通用工具
│   ├── data/            # 数据访问
│   ├── external/        # 外部服务
│   ├── security/        # 安全相关
│   ├── web/             # Web相关
│   └── index.js         # 基础设施入口
└── config/              # 配置文件
```
